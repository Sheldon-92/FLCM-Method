# FLCM 2.0 用户指南

## 快速开始

### 安装
```bash
curl -fsSL https://raw.githubusercontent.com/Sheldon-92/FLCM-Method/main/install.sh | bash
```

### 基本使用
```bash
# 创建内容
flcm create "你的主题"

# 快速模式
flcm quick "AI trends 2024"

# 查看帮助
flcm help
```

## 核心功能

- **智能学习引擎**: 基于 AI/ML 的自适应学习路径
- **知识图谱**: 自动构建知识关联网络
- **15+ 学习框架**: SWOT-USED、SCAMPER、苏格拉底提问等
- **多平台支持**: LinkedIn、Twitter/X、微信、小红书
